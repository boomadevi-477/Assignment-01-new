Fast Loop Line

For foundries with larger mould handling requirements, we developed a robust fast loop system. As well as catering for moulds up to a maximum size of 3500mm x 2500m x 880mm, the fast loop can also adapt to varying fill times, making the entire moulding process more efficient.
The modular design of the fast loop system allows easy configuration for individual customer needs while the integrated compaction table offers variable frequency and is complete with protective sand deflectors.

  • Good productivity with minimal operators
  • Standard options suitable for moulds sizes from 1000mm x 800mm up to 3500mm x 2500mm
  • Bespoke systems available up to 7000mm x 2000mm
  • Option to add an auto strickle
  • Modular design for greater flexibility
  • Modular design for greater flexibility
  • Integrated compaction table, transfer cars and stripping unit.
